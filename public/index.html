üßë‚Äçüíª Quick start


<div id="waveform">
  <!-- the waveform will be rendered here -->
</div>

<script type="module">
  import WaveSurfer from 'https://unpkg.com/wavesurfer.js@alpha'

  const wavesurfer = WaveSurfer.create({
    container: '#waveform',
    waveColor: '#4F4A85',
    progressColor: '#383351',
    url: '/examples/audio.mp3',
  })

  wavesurfer.on('interaction', () => {
    wavesurfer.play()
  })
</script>

üß© Plugins

  • Envelope
    A graphical interface to add fade-in and -out effects and control volume.
  • Record
    Records audio from the microphone and renders a waveform.
  • Regions
    Creates clickable overlays to mark regions of audio.
  • Minimap A small waveform that serves as a scrollbar for the main waveform.
  • Timeline
    Displays notches and time labels below the waveform.
  • Spectrogram
    A visual representation of the spectrum of frequencies in audio using FFT.
